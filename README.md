# SpaceIt API

AI-powered image processing API:
- Background removal
- Image upscaling
- Image editing
- Image compression
- Batch processing

## API Endpoints

- `POST /remove-bg` - Remove background from image
- `POST /upscale` - Upscale image with AI
- `POST /edit` - Edit image (brightness, contrast, etc.)
- `POST /compress` - Compress image
- `POST /batch` - Batch process multiple images
